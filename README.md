<h1>Notes</h1>
<p><b>print(type(10))</b></br>                  
Int - whole numbers,no decimal point,used for counting, indexing and calculations </br> 
<b>print(type(3.14))</b></br>                   
Float - numbers with decimal points , used for measurements, money and averages</br> 
<b>print(type(1 + 3j))</b> </br> 
Complex - numbers with real and imaginary parts, written as a + bj, j represents √-1, used in advanced math, physics and engineering </br> 
<b>print(type('Asabeneh'))</b> </br>            
String - text data, written inside single ‘’ or double “” quotes </br> 
<b>print(type([1, 2, 3]))</b>  </br>           
List - stores multiple values, ordered and changeable (mutable), written using [], used to store items like number, names, etc </br>
<b>print(type({'name': 'Asabeneh'}))</b> </br> 
Dictionary - stores data as a key : value pairs, written using {}, used to store structured data  
<b>print(type({9.8, 3.14, 2.7}))</b>  </br>    
Set - stores unique values only, unordered, written in {} without key : value pairs, used when you don’t want duplicate values </br>
<b>print(type((9.8, 3.14, 2.7)))</b> </br>      
Tulpe - look similar to lists but use (), ordered and unchangeable</p>
